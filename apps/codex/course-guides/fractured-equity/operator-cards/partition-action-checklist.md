checklist:
  - Confirm you hold a recorded interest in the property
  - Pull current title search — identify all record owners
  - Calculate your ownership percentage
  - Document all property expenses paid (taxes, insurance, repairs)
  - Engage a real property litigation attorney in the property's state
  - File petition in district court (not county court)
  - Name all co-owners as defendants
  - Request partition by sale if physical division is impractical
  - Serve all named defendants properly
  - File notice of lis pendens on property records
  - Attend any court-ordered mediation
  - If partition by sale ordered — bid at the commissioner's sale
  - Record the commissioner's deed upon winning the sale
  - File for cost recovery from other parties if applicable
commonMistakes:
  - Filing in the wrong court (must be district court in Texas)
  - Failing to name all co-owners (missing parties can vacate the judgment)
  - Not recording lis pendens (third parties can acquire interests during litigation)
  - Underestimating timeline — plan for 9-18 months
  - Not accounting for partition sale being below market (courts don't guarantee market value)
